# Controle Operacional de Qualidade - PWA

Aplicação web para gerenciar análises de qualidade de açúcar com suporte a:

- ✅ Calculadora de Cor ICUMSA (Cubeta 1 ou 4 cm)
- ✅ Calculadora de Resíduo Insolúvel em Duplicata
- ✅ Passagem de Turno por WhatsApp
- ✅ Exportação para Excel e PDF
- ✅ Light/Dark Mode
- ✅ Armazenamento Local (Offline)

## Como Usar

1. Abra [index.html](index.html) em qualquer navegador
2. Selecione a categoria de operação
3. Preencha os dados da análise
4. Use as calculadoras para os resultados
5. Exporte os dados em Excel ou WhatsApp

## Funcionalidades

### Categorias
- Navio
- Caminhão Extra
